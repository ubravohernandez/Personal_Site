  • The display property can hold only four values, all of which have different affects on the element.
    • Block: Takes up an entire line. Width and heigth can be modified.
    • Inline: The only space take is enough to fit the content. Cannot modify height and width.
    • Inline-Block: Takes the best of Block and Inline
    • None: Essentially removes the element form sight.
    Another tag related to Display:None is visibility:none. Which removes the content from site, but does not remove the space that the content takes up.
02/13/2020
  • CSS Positioning continued. CSS has a property know as position. This includes four possible values that we can affect.
    • Static: This is essentially what the web browser does normally when displaying HTML
    • Relative: This moves the position of the content relative to where it would have been if it were static. For example, position: relative; top: 50px; would move the content 50 pixels down.
    • Absolute: This allows you to change the position of a child whose parent has a relative position.
    • Fixed: This simply fixes things to the screen. (This unlocks something like a search/navigation bar!)
